# XJTLU-CS-Notes-2023-2025

This repository contains categorized notes for courses taken between 2023 and 2025 at XJTLU. The notes are organized into folders for each academic year (大一, 大二, 大三, 大四).

## Structure
- **大一**: Notes for first-year courses.
- **大二**: Notes for second-year courses.
- **大三**: Notes for third-year courses.
- **大四**: Notes for fourth-year courses.

## Purpose
The purpose of this repository is to provide a centralized location for all course-related materials, making it easier to review and reference.

## Usage
Feel free to clone this repository and navigate through the folders to find the notes you need.
